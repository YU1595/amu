# 編 / amu

孤独を感じている若者が、今の気持ちをそっと置き、同じ気持ちの人がいることを点と糸の連なりで感じられるMVPです。

## 体験

1. 今の感情を1タップで選ぶ
2. 同じ感情の人数を見る
3. 点の集まりで「一人じゃない」と感じる

## 使い方

`index.html` をブラウザで開くと起動します。

実装は静的ファイルのみです。

- `index.html`
- `styles.css`
- `app.js`

## メモ

- テキスト入力、コメント、プロフィール、いいね、ランキングはありません。
- 匿名IDと感情ログはブラウザ内に保存されます。
- PCブラウザでの操作を想定したレイアウトです。
